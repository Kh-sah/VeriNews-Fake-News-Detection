# VeriNews — AI Fake News Detection

Precision fact analysis powered by neural language intelligence.

---

## About
VeriNews is an AI-powered fake news detection system that classifies 
news articles as Real or Fake using Machine Learning.

---

## Models Used
- Logistic Regression — 99.1% Accuracy
- Naive Bayes — 94% Accuracy

---

## Project Files
- Day1_Data_Exploration.ipynb — Load and explore dataset
- Day2_Text_Preprocessing.ipynb — Clean and preprocess text
- Day3_Vectorization_and_Split.ipynb — TF-IDF and train/test split
- Day4_Model_Training.ipynb — Train models and visualize results
- index.html / style.css / script.js — Frontend UI
- app.py — Flask backend API
- requirements.txt — All required libraries

---

## Results
- Logistic Regression Accuracy : 99.1%
- Naive Bayes Accuracy         : 94%

---

## Tech Stack
- Python, Scikit-learn, NLTK, Matplotlib
- Flask Backend
- HTML, CSS, JavaScript Frontend

---

## How to Run
1. Install dependencies:  pip install -r requirements.txt
2. Run backend:  python app.py
3. Open index.html in browser
